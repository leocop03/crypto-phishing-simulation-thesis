# Crypto Phishing Simulation Thesis

This repository contains the experimental component of a Bachelor's thesis in Cybersecurity.

The project implements a controlled agent-based simulation to study how synthetic LLM-based user profiles react to phishing scenarios in cryptocurrency ecosystems.

## Purpose

The goal is not to measure real human behavior, but to build an exploratory framework for comparing different user archetypes, phishing techniques and security-awareness levels.

## Structure

- `agents/`: synthetic user archetypes
- `scenarios/`: phishing and legitimate message templates
- `simulations/`: Python scripts to run the simulation
- `analysis.ipynb`: notebook for data analysis and plots
- `results/`: generated simulation outputs, ignored by Git

## Disclaimer

This project is for educational and academic research purposes only.  
No real phishing campaign, credential harvesting, or malicious infrastructure is involved.
All scenarios are synthetic and executed only in a controlled local simulation.
